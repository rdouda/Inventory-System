# Inventory System
## Inventory System in Unreal Engine 5

### Current features
* Detecting pickupable items with raytrace
* Swapping 2 items
* Picking up and dropping items
* Saving and loading the game
* Interacting with items

### Planned features
* User Interface
* Multiplayer support
